# Security Policy

This repository contains public workflow templates and prompts. It does not contain private Ringly source code, production secrets, customer data, App Store Connect credentials, or live API keys.

## Reporting A Problem

Open a GitHub issue if you find:

- A template that encourages unsafe credential handling.
- A checklist that could cause accidental public disclosure.
- A workflow that tells an agent to test, scan, or modify systems it does not own.
- A script that could run destructive commands without clear operator intent.

Do not publish real secrets in an issue. Replace them with placeholders and describe the exposure path.

## Scope

Security review for this repo is limited to the public workflow materials here. Do not use these templates as permission to scan, probe, or review private repositories or systems you do not own.
